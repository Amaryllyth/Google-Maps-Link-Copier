# Google Maps Link Copier

A simple Chrome extension that automatically extracts the original place name and short sharing link from Google Maps. With a single click or keyboard shortcut (`Ctrl+Shift+C`), it formats the information into a Markdown link (e.g., `[Place Name](https://goo.gl/...)`) and copies it directly to your clipboard.

## How to use

1. **Install the extension**: Load this unpacked extension folder in your browser via `chrome://extensions/`.
2. **Find a place**: Open Google Maps and click on a specific place to open its details panel.
3. **Copy the link**: 
   - Click the extension icon in your browser toolbar.
   - Or use the keyboard shortcut `Ctrl+Shift+C` (Mac: `Cmd+Shift+C`).
4. **Done!** A Markdown link (e.g., `[Place Name](https://goo.gl/...)`) is now copied to your clipboard, and an alert will confirm the action.
